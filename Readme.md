# Profile API — Stage 1 Backend Assessment

## 📌 Overview

This project is a RESTful API that accepts a name, retrieves demographic data from external APIs, classifies the results, stores them in a MongoDB database, and exposes endpoints to manage the stored profiles.

The API integrates with:

- Genderize API
- Agify API
- Nationalize API

It applies classification logic and prevents duplicate records.
